# 2025_2_ict-final-overgrown
정문기입 기말 프로젝트 git 저장소

## 개요
- 프로젝트명: 빛과 그림자의 정원
- 장르: 인터랙티브 미디어 아트 게임
- 개발 환경: JavaScript (p5.js Library)
- 기획 배경
    - 모티브: 인적이 끊겨 자연에 잠식당한 폐허의 이미지
    - 의도: 디지털 공간에서도 시간이 흐르고(이끼), 사용자의 행동(빛)에 따라 반응하는 살아있는 정원의 구현

## 핵심 기능 및 구현
- User Input
    - Mouse Interaction: 움직임(광원 이동), 우클릭(씨앗 심기)
    - Keyboard Event: 스페이스바(화면 캡처), 'R/ㄱ'키(리셋), ESC(강제종료 후 스토리라인 전환)
- Main Controller
    - 상태관리 시스템 -(초기화 및 제어)-> 객체 관리자 -(데이터 참조)-> 핵심 알고리즘
    - 게임 오브젝트(Light, Moss, Plant)
- Server
    - 서버/아카이브, Supabase QR

- 게임 상태 관리
    - Flow: 시작/튜토리얼 -> 플레이 -> 스토리라인 -> 엔딩 -> 타이틀
    - Transition:
        - 트리거: 이끼의 화면 점유율 90% 달성 시 Delay Timer 가동 -> 2초 뒤 스토리라인으로 자동 전환
        - 연출: 전환 직전 화면 캡처
- 핵심 알고리즘
    - 이끼:
        - 게임 주기에 따라 이끼 생성 속도 및 빈도 조절
        - 완전히 생성된 이끼가 식물에 닿으면 3초 타이머 작동, 3초 넘으면 식물 파괴
        - 빛에 닿으면 이끼 정화되고, 이끼의 화면 점유율 90% 넘으면 게임 오버
- 객체 제어
    - 객체 생명주기 관리:
        - 생성: 조건에 따른 객체 추가
        - 제거: 수명이 다한 객체 배열에서 삭제
    - 상호작용 제어:
        - 광원-이끼/이끼-식물 충돌 감지

## 디자인
- Assets
    - 다양한 크기 및 종류의 꽃, 빛 구체, 잎사귀, 배경화면 등 assets 별도 제작
    - 부드럽고 귀여운 게임 분위기를 조성할 수 있도록 색연필로 그린 듯한 그림체로 디자인
- Game State별 UI
    - 각 게임 단계 배경 디자인
        - 시작, 튜토리얼, 플레이, 엔딩, 스토리라인 등
        - 직접 그린 배경화면 asset 또는 화면 스크린샷 + 텍스트 or 텍스트 애니메이션
    - 인터랙션 항목 디자인
        - 마우스오버 효과, ‘CLICK ANYWHERE ON SCREEN’ 안내 버튼 등
    - 위 두 디자인을 통합하여 UI 함수로 작성

## 디벨롭 과정
- 게임에 대한 이해를 증진할 수 있도록 직관적인 튜토리얼을 제작했으면 좋겠음
    - 튜토리얼 내용 구체화
    - 길어진 분량의 튜토리얼을 두 개의 슬라이드로 분화
        - 마우스 클릭을 통해 슬라이드 넘김
    - 튜토리얼 내 assets 이미지 삽입하여 직관적 이해 도움
- 게임의 목적성, 또는 방향성이 불분명하다. 왜 이 게임을 해야 하는지 이유가 명확하지 않음
    - 스토리라인 구상 후 추가
        - ESC 키를 눌러 게임 플레이를 중단하려는 플레이어에게: “빛과 그림자의 정원에 잠시나마 들러 주어서 감사합니다”
        - 엔딩 화면(이끼의 잠식)에 도달한 플레이어에게: “당신의 여정에 함께할 수 있어 영광이었습니다”
    - 텍스트 애니메이션 적용 (타자기 효과) + 마우스 클릭을 통해 텍스트 넘김
- 게임 중 이끼 로직과 관련된 피드백
    - 식물의 의문사가 너무 많음
        - 완전히 생성된 이끼만 충돌하게 하고, 충돌 후 식물 파괴 타이머 추가
    - 이끼가 화면 아래에서도 자라는건 너무 불합리함
        - 화면 아래 이끼 생성 삭제
    - 이끼 정화 난이도가 너무 높음
        - 주기(특히 밤)에 따른 이끼 생산 속도 늦춤
    - 이끼 과성장 시작되면 렉이 너무 심하게 걸림
        - 과성장시 이끼 전체 수 줄이고, 미감을 해치지 않는 선에서 간격 조절
- 스페이스바 저장 될 때도 있고 안 될 때도 있음
    - 저장은 계속 됐지만, 투명도로 인해 보이지 않았음 -> 알파값 255로 초기화되도록 수정
- 튜토리얼을 다시 보고 싶음
    - 튜토리얼로 돌아갈 수 있는 버튼 생성
